# API Reference

Здесь вы найдете документацию по всем API сервисам, используемым в KIRA AI.

## Как использовать API документацию

Документация автоматически генерируется из исходного кода в директории `src/services/`.
Для каждого сервиса создается отдельная страница с описанием всех экспортируемых функций,
их параметров и типов данных.

## Доступные сервисы

API документация будет автоматически обновляться при изменении сервисов.
