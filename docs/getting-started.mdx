# Начало работы с KIRA AI

## Регистрация

1. Перейдите на страницу регистрации
2. Введите email и пароль
3. Подтвердите email по ссылке из письма

## Первые шаги

После регистрации вы будете перенаправлены на дашборд, где можно:

- Создать первую заметку
- Запланировать задачу
- Начать диалог с AI-ассистентом
- Синхронизировать Google Calendar

## Роли пользователей

В системе существует две основные роли:
- **USER** — стандартная роль для большинства пользователей
- **SUPER_ADMIN** — роль с расширенными возможностями администрирования
