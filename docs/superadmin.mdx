# Роль SUPER_ADMIN

Роль с расширенными административными возможностями.

## Доступные возможности

- Все возможности обычного пользователя
- Доступ к административной панели
- Управление пользователями и их данными
- Мониторинг использования системы
- Настройка глобальных параметров
- Управление интеграциями и API-ключами
- Доступ к расширенной аналитике
