# Роль USER

Стандартная роль для всех пользователей платформы KIRA AI.

## Доступные возможности

- Создание и редактирование личных заметок
- Управление задачами
- Интеграция с календарем
- Общение с AI-ассистентом
- Использование доступных MiniApps

## Ограничения

- Нет доступа к административной панели
- Нет возможности управлять другими пользователями
- Лимиты использования AI зависят от тарифного плана
