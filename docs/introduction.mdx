# KIRA AI

KIRA AI — это SaaS-платформа, предоставляющая AI-ассистента с интегрированными задачами, 
заметками, календарем и расширяемыми MiniApps.

## Основные возможности

- **AI-ассистент** — интеллектуальная помощь в работе с данными
- **Заметки** — система заметок с поддержкой Markdown и связями как в Obsidian
- **Задачи** — эффективное управление задачами
- **Календарь** — интеграция с Google Calendar
- **MiniApps** — расширяемая система приложений

## Архитектура

Платформа построена на React с использованием TypeScript и предлагает современный,
гибкий и модульный интерфейс, вдохновленный ClickUp и Obsidian.
